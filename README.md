#  Flatty Landing Page Clone (SYMB_Assignment)

This project is a static, fully mobile-responsive landing page designed to replicate the clean, modern look of a flat-style UI template (named "Flatty"). It is built using pure HTML and CSS.

##  Live Deployment

The final, responsive version of this landing page is hosted on Netlify:

**[View Live Site](https://symb-assignment.netlify.app/)**

##  Repository Structure

* **`templates/`**: Contains the main source files (`index.html`, `index.css`).
* **`images/`**: Contains all visual assets used across the page.
